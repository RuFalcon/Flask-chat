# Flask chat

Простой чат на Flask с помощью Pusher

![Flask chat](http://gamedz.ru/wp-content/uploads/2020/chat.gif "Flask chat")

## Зависимости
- Flask
- Pusher
- SQLAlchemy

## Установка

Установка виртуального окружения

`python -m venv venv`

Установка зависимостей

`pip install -r requirements.txt`

## Запуск программы

`flask run`
